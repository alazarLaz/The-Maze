The Maze Project

A 3D Maze game developed using SDL2 library and C programming language

Play Game

Key Features

A Maze with diffrent textures

The game has a unique texture selection.

Realistic 3D Graphics

Experience a fully immersive environment with realistic graphics.

Simple Controls

The controls for navigating the 3D maze are simple and intuitive, making it easy for anyone to enjoy the game. Players can move forward and backward using the up and down arrow keys or the W and S keys, and can turn the camera around using the right and left arrow keys or the A and D keys.

About

I wanted to challenge myself by creating a 3D maze game using the SDL2 library and the C programming language, which allowed me to learn new skills and gain a deeper understanding of game development.

This project was completed as part of my portfolio for ALX Software Engineering School, a rigorous software engineering program that has helped me develop the skills and knowledge I need to succeed as a programmer. You can find the code for this project and other portfolio projects on my GitHub page.
